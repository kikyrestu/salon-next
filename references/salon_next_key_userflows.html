F1 Komisi Selling By — alur di POS
Kasir→ Buka POS→ Tambah item service→ Pilih "Dijual oleh"→ Checkout→ Invoice tersimpan + sellingByCommission→ ✓ Payroll ter-split
F5 WA Appointment Reminder — alur otomatis
Cron (15 min)→ Cek appointment dalam X menit→ reminderSent = false?→ Build WA message dari template→ Fonnte sendWhatsApp→ Mark reminderSent = true→ ✓ Customer ternotifikasi
F6 Kirim Nota WA — 2 path
Kasir→ Selesai transaksi→ Klik "Kirim ke Customer" / "Kirim ke Admin"→ POST /invoices/[id]/send-wa→ Build nota text dari template→ Fonnte → nomor tujuan→ ✓ Nota terkirim
F7 History Portal — alur akses customer
Customer→ Terima link WA nota→ Klik link portal/{token}→ Server cari Customer by token→ Render: membership, poin, paket, invoice→ ✓ Read-only, no login
F8 Thermal Print — alur WebUSB
Kasir→ Klik "Print Thermal"→ Browser request USB device→ Fetch /thermal binary dari server→ transferOut ke printer endpoint→ Auto cut→ ✓ Nota tercetak
⚠ Fallback: kalau WebUSB tidak support → window.print() dengan CSS @media print dioptimasi
F11 Edit / Delete Paket — Super Admin only
Super Admin→ Buka Customer Detail→ Lihat tombol Edit/Delete di paket aktif→ Edit: ubah expiry / sisa quota→ PUT /customer-packages/[id]→ checkPermission packages.delete→ ✓ Tersimpan
Kasir/staff: tombol tidak tampil (RBAC checked via usePermission hook)
